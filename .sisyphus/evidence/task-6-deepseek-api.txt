## Key Design Decisions
- Retry via tenacity: stop_after_attempt(max_retries), wait_exponential(1,1,10)
- Retries on: DeepSeekRateLimitError, DeepSeekServerError, httpx.TimeoutException
- Auth errors (401) are NOT retried (fail fast)
- Token accumulation via _total_tokens_used dict
- Cost: $0.27/MTok input + $1.10/MTok output (DeepSeek V3)
- PromptLoader uses Jinja2 FileSystemLoader from config/personas/prompts/
- Pipeline protocols use typing.Protocol (structural subtyping, no inheritance required)
