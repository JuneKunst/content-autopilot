## Implementation Notes
- YouTubeCollector uses YouTube Data API v3
- Quota tracking: SEARCH_QUOTA_COST=100, VIDEOS_QUOTA_COST=1
- Daily quota limit: 9500 (buffer from 10000 free tier)
- Two strategies: mostPopular (regionCode=KR) + search by query
- Graceful empty return when youtube_api_key is not set
- Deduplication by external_id (youtube_{video_id})
- RateLimiter: 30 requests/minute
