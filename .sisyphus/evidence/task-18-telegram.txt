## Implementation Details

### TelegramPublisher Class
- __init__: Accepts optional bot_token and channel_id, falls back to settings
- _format_message: Formats ArticleDraft as Telegram HTML message with:
  - Bold title
  - Summary (first 200 chars) or content fallback
  - Ghost URL link (if provided)
  - Source attribution link
  - Hashtags from tags (first 5, spaces replaced with underscores)
- publish: Async method that sends message to Telegram channel via Bot API
  - Returns PublishResult with status (success/failed/skipped)
  - Handles missing credentials gracefully
  - Catches HTTP errors and generic exceptions
